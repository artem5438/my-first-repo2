# Перечень используемых ресурсов и технологий в этой работе:

---

## Список пополняется...

* Создание ER-диаграмм - [dbdiagram.io](https://dbdiagram.io/d)
* Создание BPMN-диаграмм - [bpmn.io](https://bpmn.io)
* Репозиторий кода, а также DevOps-платформа - [github.com](https://github.com)
* Система управлениями версиями - [git](https://git-scm.com/)
* Серверная часть проекта - [djangoproject.com](https://www.djangoproject.com/)
* База данных для сервера - [postgresql.org](https://www.postgresql.org/)
* Контейнеризация - [docker.com](https://docker.com)
* Интерактивная разработка концепта проекта - [micro.com](https://miro.com/)
